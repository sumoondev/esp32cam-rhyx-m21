# ESP32CAM RHYX-M21 

**Just trying and testing Esp32 cam module with RHYX-M21 cam and tryping to identify, benchmark, and optimize the camera modules.**

## Quickstart: flash & run the init-matrix test (Arduino)
1. Clone this repo:
   ```bash
   https://github.com/sumoondev/esp32cam-rhyx-m21.git
   cd esp32cam-rhyx-m21
    ```
2. Open in Arduino IDE (or import into PlatformIO).
3. Configure serial baud and board type (AI-Thinker/your-board) in the sketch if needed.
4. Flash to the ESP32 and open Serial Monitor at 115200.
